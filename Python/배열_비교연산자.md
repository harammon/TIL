# 비교 연산자로 배열 비교하기

- 배열의 대소 또는 등가 관계는 비교 연산자를 사용하여 판단한다.

  ~~~ python
  >>> [1, 2, 3] == [1, 2, 3]
  >>> True
  >>> [1, 2, 3] < [1, 2, 4]
  >>> True
  >>> [1, 2, 3] < [1, 2, 3, 1]
  >>> True
  >>> [1, 2, 3] < [1, 2, 3, 1] < [1, 2, 3, 4]		# and 결합
  >>> True
  >>> [1, 2, 3] < [0, 0, 0, 0]
  >>> False
  ~~~

- 맨 앞 원소부터 차례로 비교하며, 원소의 값이 같으면 다음 원소를 비교한다.

  -> 만약 어느 원소의 값이 크면, 그 배열이 큰 것으로 판단한다.

- 배열의 원소 수가 다른 경우에는, 원소 수가 많은 배열을 더 크다고 판단한다. (이전 비교 대상까지 모두 같았을 경우)
